# The Archive of Civilization

A five-page static website designed for GitHub Pages. The project contrasts polished elite history with fragmented records of anonymous labor.

## Pages

- `index.html` - official archive homepage
- `figures.html` - great figures / canonical biographies
- `workers.html` - unnamed workers / fragmented labor records
- `projects.html` - historic projects and their hidden labor
- `missing-records.html` - blank archive room / manufactured absence

## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder.
3. In the repository settings, open `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and `/ (root)` folder.
6. Save, then wait for the site URL to appear.

If you want a user site, name the repo `<your-github-username>.github.io`.
If you want a project site, any repo name works and the URL will be `https://<your-github-username>.github.io/<repo-name>/`.
